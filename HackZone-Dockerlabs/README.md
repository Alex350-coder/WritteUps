
# README — HackZone Dockerlabs (ES / EN)

## ES — Resumen

Este repositorio contiene la documentación del laboratorio "HackZone", diseñada para ejercicios de enumeración, explotación de cargas de archivos y escalada de privilegios en un entorno controlado. Los archivos principales son:

- `HackZone.md`: Descripción detallada del recorrido y análisis en español.
- `HackZoneEn.md`: Traducción profesional al inglés del documento anterior.
- `QuickStart.md`: Resumen rápido y bilingüe con comandos esenciales para completar el laboratorio en pocos minutos.
- `Img/`: Carpeta con imágenes ilustrativas utilizadas en los documentos.

## Uso

1. Revisar `QuickStart.md` para obtener los comandos básicos y la secuencia de trabajo.
2. Seguir las secciones de `HackZone.md` para un análisis paso a paso y las recomendaciones de mitigación.

## Precauciones

- Ejecutar únicamente en entornos de laboratorio autorizados.
- No reutilizar credenciales reales ni almacenar secretos en texto plano.

## EN — Summary

This repository contains the documentation for the "HackZone" lab, intended for exercises in enumeration, file upload exploitation and privilege escalation in a controlled environment. Main files:

2. Follow `HackZoneEn.md` or `HackZone.md` for the full walkthrough and mitigation recommendations.
