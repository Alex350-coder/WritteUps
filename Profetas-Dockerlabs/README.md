# Profetas — Dockerlabs

Español / Spanish:

Este repositorio contiene la documentación del laboratorio "Profetas". El objetivo es practicar técnicas de reconocimiento, explotación web (SQLi, XXE), acceso por SSH y escalada de privilegios hasta `root`. Consulte `Profetas.md` para la guía completa, `QuickStart.md` para pasos rápidos y `ProfetasEN.md` para la versión en inglés.

Recomendado: siga el `QuickStart` para ejecutar las pruebas mínimas y use herramientas como `nmap`, `gobuster`, `Burp Suite` y utilidades de decompilación para `.pyc`.


English / Inglés:

This repository documents the "Profetas" lab. The exercise focuses on reconnaissance, web exploitation (SQLi, XXE), SSH access and privilege escalation to `root`. See `Profetas.md` for a full walkthrough, `QuickStart.md` for quick steps, and `ProfetasEN.md` for the English version.

Recommended tools: `nmap`, `gobuster`, `Burp Suite` and Python decompilation utilities.
